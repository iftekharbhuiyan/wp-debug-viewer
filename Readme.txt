== Description ==

A Plugin that allows site Administrators to see debug log content from Dashboard.


== Installation ==

1. Upload the `debug-viewer` folder to your site's `/wp-content/plugins/` directory.
2. Activate the plugin from the Plugins screen in WordPress.
